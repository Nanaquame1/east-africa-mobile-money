# east-africa-mobile-money

\# East Africa Mobile Money Analysis (2017–2024)



An end-to-end analysis of mobile money growth, financial inclusion impact,

and the agent network expansion across Kenya, Tanzania, Uganda, Rwanda

and Ethiopia — using GSMA and World Bank data.



\## Key Findings



\- Region grew from \*\*71.6M to 162.9M registered accounts\*\* (2017–2024)

\- \*\*Kenya's M-Pesa contributes 61% of GDP\*\* — most integrated mobile money economy globally

\- \*\*Ethiopia's Telebirr grew from 3.2M to 45.8M accounts in 3 years\*\* — fastest growth in region

\- \*\*Kenya's unbanked population fell from 17% to 8%\*\* directly tied to mobile money expansion

\- \*\*Agent network grew 147%\*\* to 1.1M agents — the last-mile infrastructure backbone
